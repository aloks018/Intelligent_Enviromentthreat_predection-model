# IEIRAS Premium Frontend

Modern React homepage for the IEIRAS environmental intelligence platform.

## Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
- Recharts

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Notes

- This frontend lives alongside the existing Streamlit project and does not replace it automatically.
- Deploy this folder separately on Vercel, Netlify, or Render Static Sites for the premium SaaS homepage experience.
